# Photos & images

Place image files in this `public/` folder, then reference them in code by
their path starting with `/`. For example, a file at `public/mayor.jpg` is
used as `/mayor.jpg`.

Suggested images to add:
- `mayor.jpg`        — portrait of Mayor Shirley (used in Hero + About)
- `holly-springs.jpg`— a city/civic photo for backgrounds
- `og-image.jpg`     — 1200×630 social-share preview image

See the project README ("Adding photos") for the exact code snippet to swap a
placeholder for a real photo.
